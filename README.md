## 📖 Project Overview
This repository contains a technical tutorial on the **Vanishing Gradient Problem** in Deep Multilayer Perceptrons (MLPs). Using the **UCI Dry Bean Dataset**, the tutorial demonstrates how to diagnose gradient decay and implement professional-grade solutions using PyTorch.

## 📊 Key Findings
- **The Problem:** Sigmoid activation functions cause a significant drop in learning signals by the earlier layers.
- **The Solution:** Implementing **ReLU** combined with **He-Initialization** restores gradient flow and increases classification accuracy significantly.
